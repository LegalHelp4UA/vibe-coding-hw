# vibe-coding-hw
_Старт навчання: червень 2026._

Навчальний репозиторій (Web Academy «Вайб-кодинг», ДЗ Уроку 1 — основи Git).

## Ідея проєкту
Міні-помічник «Чекліст документів для нотаріального оформдення та апостиля»: підказує, які документи
та кроки потрібні людині, що оформлює нотаріальні документи в іноземного нотаріуса та апостиль. Поки що — лише опис;
код додаватиму протягом курсу.

## Цільова аудиторія
Українці за кордоном, яким треба підготувати нотаріальні документи з апостилем.

## Заплановані функції
- Вибір типу документа (довіреність, згода, заява на спадщину, інший документ).
- Перелік кроків і орієнтовних строків.
- Двомовні підказки (укр / англ).
- Посилання на офіційні джерела щодо апостилю та нотаріату.

> Це навчальний проєкт. Не є юридичною консультацією.
