# LegalHelp4UA — інформаційний Telegram-бот

LegalHelp4UA — навчальний Telegram-бот на Python та aiogram 3.x.

Проєкт поступово розвиватиметься у клієнтського бота-консультанта для сервісу LegalHelp4UA. На поточному етапі бот надає базову інформацію про послуги, процес оформлення документів і способи зв'язку.

## Цільова аудиторія

Бот створюється для українців за кордоном, яким потрібно дистанційно оформити нотаріальні документи з апостилем.

## Команди бота

- `/start` — привітання
- `/services` — які документи оформлюємо
- `/process` — як відбувається оформлення
- `/contacts` — як зв'язатися
- `/help` — список команд

## Запуск локально

1. Створіть віртуальне середовище:

```powershell
python -m venv .venv
```

2. Активуйте його у Windows PowerShell:

```powershell
.venv\Scripts\Activate.ps1
```

3. Встановіть залежності:

```powershell
pip install -r requirements.txt
```

4. Створіть файл `.env` на основі `.env.example` і додайте `BOT_TOKEN`, отриманий від [@BotFather](https://t.me/BotFather).

5. Запустіть бота:

```powershell
python -m app.main
```

## Запуск через Docker з hot reload

Запустіть проєкт через Docker Compose:

```bash
docker compose up --build
```

Зміни у директорії `app/` автоматично перезапускають бота.

## Перевірка секретів перед комітом

Перед комітом запустіть перевірку:

```powershell
python scripts/security_check.py
```

## Безпека

Реальний `BOT_TOKEN` має зберігатися тільки у файлі `.env`, який додано до `.gitignore`.

Не комітьте `.env` і не додавайте реальний токен у код, README або інші файли репозиторію. У репозиторії має бути лише `.env.example` з плейсхолдером.

## Roadmap

Цей навчальний бот є основою майбутнього клієнтського бота-консультанта LegalHelp4UA.

Майбутній розвиток може включати:

Поки що проєкт перебуває на навчальному етапі.
